    mysqladmin kill¶

    • https://tech.basicinc.jp/articles/41
    • https://stackoverflow.com/questions/44192418/how-to-find-mysql-process-list-and-to-kill-those-processes

    処理時間に60秒以上かかっているのを表示¶

    SELECT * FROM information_schema.PROCESSLIST WHERE TIME > 59;

    指定ユーザのプロセスIDのを表示¶

    SELECT GROUP_CONCAT(ID) FROM information_schema.PROCESSLIST WHERE user='user';

    処理時間に60秒以上かつ指定ユーザのプロセスIDのを表示¶

    SELECT GROUP_CONCAT(ID) FROM information_schema.PROCESSLIST WHERE user='user' and TIME > 59;

    結果¶

    SELECT GROUP_CONCAT(ID) FROM information_schema.PROCESSLIST WHERE  user='user' and TIME > 59;
+------------------+
| group_concat(ID) |
+------------------+
| 1,2,3,4,5        |
+------------------+

    複数のプロセスIDをkillする¶

    $ mysqladmin kill 1,2,3

    killコマンドを出力する¶

    SELECT  concat('mysqladmin kill ', GROUP_CONCAT(ID)) FROM information_schema.PROCESSLIST WHERE user='user';

+--------------------------------------------------------------------------------------------+
| concat('mysqladmin kill ', GROUP_CONCAT(ID))                                               |
+--------------------------------------------------------------------------------------------+
| mysqladmin kill 588732677,326878732692,588732693,588732695,588732684,588732696,588732688   |
+--------------------------------------------------------------------------------------------+

[root@mysqlxxx ~]# mysql -e "SELECT  concat('mysqladmin kill ', GROUP_CONCAT(ID)) FROM information_schema.PROCESSLIST WHERE user='user';"
