Smart表

項目ID 項目名 詳細な説明
01 Raw Read Error Rate この項目はハードディスクからデータを読み込む時に発生したエラーの割合を表す。数値が閾値より低い場合、ハードディスク内の磁気ディスクまたは磁気ヘッドに異常がある。
05 Reallocated Sectors Count 代替処置（データを特別に予約した予備エリアに移動する）を施された不良セクタの数。
C5(197) Current Pending Sector Count 現在異常があり、代替処理を待つセクタの総数。もし後で読み込みに成功したセクタがあれば、この値は減少する。
C6(198) Off-Line Scan Uncorrectable Sector Count オフラインスキャン時に発見された、回復不可能なセクタの総数。この値が増加する場合は、磁気ディスクの表面に明確な問題がある。 
Media Error代表扇区有问题，坏道等；
Other Error代表磁盘可能有松动。

MegaRAIDが認識しているHDDを全表示する方法

$ sudo MegaCli64 -PDList -aALL
$ sudo MegaCli64 -CfgDsply -aALL   ← RAIDカードも表示する。

HDDをリビルドする手順

1.
#対象ディスクのオフライン
/opt/MegaRAID/MegaCli/MegaCli64 -PDOffline -PhysDrv[252:0] -a0
Adapter: 0: EnclId-252 SlotId-0 state changed to OffLine.
Exit Code: 0x00

2.
#status確認
/opt/MegaRAID/MegaCli/MegaCli64 -PDList -a0 | less
Firmware state: Offline  ←　offlineになってる。

3.
#対象のディスクにmissing markつける
/opt/MegaRAID/MegaCli/MegaCli64 -PDMarkMissing -PhysDrv[252:0] -a0
EnclId-252 SlotId-0 is marked Missing.
Exit Code: 0x00

4.
#missing markのついたdiskの確認
/opt/MegaRAID/MegaCli/MegaCli64 -PDGetMissing -aALL
    Adapter 0 - Missing Physical drives
    No.   Array   Row   Size Expected
    0     0       0     1907200 MB

5.
#削除前の準備
/opt/MegaRAID/MegaCli/MegaCli64 -PDPrpRmv -PhysDrv[252:0] -a0
Prepare for removal Success
Exit Code: 0x00

6.
#Missing markの状態変更
/opt/MegaRAID/MegaCli/MegaCli64 -PDReplaceMissing -PhysDrv[252:0] -Array1 -row0 -a0
 
7.
#Rebuildスタート
/opt/MegaRAID/MegaCli/MegaCli64 -PDRbld -Start -PhysDrv[252:0] -a0

8.
#状態確認
/opt/MegaRAID/MegaCli/MegaCli64 -PDList -a0 | less
Firmware state: Rebuild   ← rebuildのステータスになった

9.
#Rebuildの状況確認
/opt/MegaRAID/MegaCli/MegaCli64 -PDRbld -showprog -physdrv[252:0] -a0
Rebuild Progress on Device at Enclosure 252, Slot 0 Completed 4% in 11 Minutes.
Exit Code: 0x00

キャッシュ設定方法

RAIDカードの書き込みキャッシュの設定 

＜Write スルー＞
MegaCli -LDSetProp WT -Lall -Aall
＜Write バック＞
MegaCli -LDSetProp WB -Lall -Aall

RAIDカードの読み込みキャッシュの設定 

＜No read ahead＞
MegaCli -LDSetProp NORA -Lall -Aall
＜Read ahead＞
MegaCli -LDSetProp RA -Lall -Aall
＜Adaptive read ahead＞
MegaCli -LDSetProp ADRA -Lall -Aall

RAIDカードのキャッシュポリシーの設定 

＜Chached＞
MegaCli -LDSetProp -Cached -Lall -aALL
＜Direct＞
MegaCli -LDSetProp -Direct -Lall -aALL

BBUユニットが不良の場合に書き込みキャッシュを使うかどうか 

＜CachedBadBBU：書き込みキャッシュ有効＞
MegaCli  -LDSetProp -CachedBadBBU -Lall -Aall
＜NoCachedBadBBU：書き込みキャッシュ無効＞
MegaCli  -LDSetProp -NoCachedBadBBU -Lall -Aall

ドライブのキャッシュを有効にするかどうか 

＜Enables drive cache：有効＞
MegaCli -LDSetProp -EnDskCache -Lall -Aall
＜Disables drive cache：無効＞
MegaCli -LDSetProp -DisDskCache -Lall -Aall

Usage

# cd /opt/MegaRAID/storcli
# ./storcli64

  • information about every controller

    • ./storcli64 show

  • informations to selected controller and its configuration

    • ./storcli64 /c0 show

  • informations to the available hard drives and their current state (Serial Number, temperature )

    • ./storcli64 /c0 /eall /sall show [all]

  • informations to the available virtual drives and their current state.

    • ./storcli64 /c0 /vall show [all]

  • show the progress of ongoing rebuilds

    • ./storcli64 /c0 /eall /sall show rebuild

  • create a RAID 1

    • ./storcli64 /c0 add vd type=raid1 size=10gb names=tmp1 drives=0:2-3

  • create a RAID 5

    • ./storcli64 /c0 add vd type=raid5 size=10gb names=tmp1 drives=0:2-6

  • create a RAID 6

    • ./storecli64 /c0 add vd type=raid6 size=10gb names=tmp1 drives=0:2-7

  • create a RAID 10

    • ./storecli64 /c0 add vd type=raid10 size=2gb,3gb,4gb names=tmp1,tmp2,tmp3 drives=0:2-3,5,7 pdperarray=2
