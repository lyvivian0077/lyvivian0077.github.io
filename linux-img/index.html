    1. 最新のRocky8/Rocky9のISOファイルをダウンロードする

8. iptablesの設定
    [root@92-tmp ~]# cat /root/bin/iptables.sh
    #!/bin/sh

    set -e

    iptables -F
    iptables -X
    iptables -P INPUT ACCEPT
    iptables -P OUTPUT ACCEPT
    iptables -A INPUT -i lo -j ACCEPT
    iptables -I INPUT -p icmp --icmp-type 0 -j ACCEPT
    iptables -I INPUT -p icmp --icmp-type 8 -j ACCEPT
    iptables -A INPUT -m state --state ESTABLISH,RELATED -j ACCEPT

    iptables -A INPUT -s 172.16.0.0/12 -j ACCEPT
    iptables -A INPUT -s 192.168.0.0/16 -j ACCEPT

    iptables -A INPUT -j DROP

    /sbin/iptables-save > /etc/sysconfig/iptables

    [root@92-tmp ~]# /root/bin/iptables.sh

    11. HVのVLANを削除したい場合

      この作業は慎重に実施ください 
      1. ブリッジ確認
   - `brctl show`
2. ブリッジ配下のIF vlan15 を削除
   - `brctl delif br0 vlan15`
3. 該当ブリッジをdown
   - `ip link set br0 down`
4. 該当ブリッジを削除
   - `brctl delbr br0`
5. ブリッジ確認
   - `brctl show`

**仮装インタフェースの削除**

1. vlanインタフェースの確認
   - `ip a | grep vlan`
   - `ip -s -d addr show dev vlan15`

2. 該当インタフェースをシャットダウン
   - `ip link set dev  vlan15 down`

3. 該当インタフェースを削除
   - `ip link delete vlan15@bond0`  <-- @bond0は必要ない。
   - `ip link delete vlan15`

4. vlanインタフェースの確認
   - `ip a | grep vlan`
